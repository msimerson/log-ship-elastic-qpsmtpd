# Ship Qpsmtpd Logs to Elasticsearch

Reads qpsmtpd JSON-formatted log files and saves them to Elasticsearch.

# Install


```sh
npm i log-ship-elastic-qpsmtpd
```

Edit log-ship-elastic-qpsmtpd.ini, then launch with:

```sh
node server.js
```
